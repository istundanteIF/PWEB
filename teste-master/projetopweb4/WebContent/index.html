Projeto GitHub

Passos para conectar o GitHub com Eclipse

  1. Instalar e configurar o Git no seu computador
  2. Acessar o site do GitHub
    • Criar um usuário no GitHub
    • Criar um repositório no GitHub
  3. Clonar o repositório
    • Clicar em "Clone a Git repository"
    • Copiar e colar a URI
    • Se a senha não funcionar, criar e utilizar um token
  4. Enviar o commit de alteraçôes para o GitHub
    • Clicar em Team > Share Project...
    • Selecionar o repositório
    • Realizar o commit selecionando os arquivos
